Design System

无为设计系统

WuWei Design System · 不泥于古，不惑于今

以中国传统哲学「无为」为精神内核，融合 2026 年设计趋势——液态玻璃、空间感、AI 对话、功能性微动效——构建的现代 UI 组件库。
HTML + Tailwind CSS，copy-paste 即用。

快速开始

复制你需要的组件代码即可使用。所有组件基于 Tailwind CSS 3.x + design-system.css。

tailwind.config.js 扩展 
module.exports = {
  theme: {
    extend: {
      colors: {
        ink: '#1a1a1b',           // 墨色 - 主文字
        paper: '#f9f6f1',         // 纸张色 - 主背景

滚动揭示 Scroll Reveal

IntersectionObserver 实现元素渐现。用 reveal 类标记，滚动进入视口时 visible 类被添加。

元素 1
元素 2
查看代码 
<!-- CSS -->
.reveal { opacity:0; transform:translateY(40px);
  transition: opacity .8s cubic-bezier(.16,1,.3,1),
              transform .8s cubic-bezier(.16,1,.3,1); }
.reveal.visible { opacity:1; transform:translateY(0); }

<!-- HTML -->
<div class="reveal">内容</div>
<div class="reveal reveal-delay-1">延迟0.1s</div>

<!-- JS -->
const observer = new IntersectionObserver((entries) => {
  entries.forEach(e => {
    if (e.isIntersecting) e.target.classList.add('visible');
  });
}, { threshold: 0.1 });
document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
